
# TransBnk Risk Radar API Documentation

## Overview
Risk Radar is a flagship product by **TransBnk** that provides a unified view of real-time balances and account statements across multiple banks under a single login. It simplifies corporate banking operations, ensures secure API integration, and supports onboarding, account linking, and statement retrieval.
